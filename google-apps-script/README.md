# Apps Script 연결 방법

1. Google Sheet에서 `확장 프로그램 > Apps Script`를 연다.
2. `Code.gs` 내용을 이 폴더의 `Code.gs` 내용으로 교체한다.
3. `배포 > 새 배포 > 유형 선택 > 웹 앱`을 선택한다.
4. 실행 권한은 `나`, 액세스 권한은 `모든 사용자`로 둔다.
5. 배포 후 나온 웹 앱 URL을 복사한다.
6. `src/config.js`의 `defaultAppsScriptUrl` 값에 복사한 URL을 넣는다.

학번 앞 두 자리는 자동으로 탭에 매핑된다.

- `11` -> `1-1`
- `12` -> `1-2`
- `13` -> `1-3`
- `14` -> `1-4`
- `15` -> `1-5`

이미 값이 들어간 `1.1`부터 `7.2`까지의 점수 칸은 다시 덮어쓰지 않는다.
